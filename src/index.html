DSE API - Available Routes

Route HTTP Method Description Examples
/live GET Get live stock data for all stocks (cached for 1 minute) http://localhost:3000/live- Live data for all stocks
/live/:code GET Get live data for a single stock by code http://localhost:3000/live/GENNEXT- Live data for stock code GENNEXT
/dse30 GET Get data for DSE30 shares http://localhost:3000/dse30- Live data for DSE30 stocks
/top20 GET Get data for Top 20 shares http://localhost:3000/top20- Live data for top 20 shares
/archive GET Get archived stock data between dates (query params: startDate, endDate) http://localhost:3000/archive?startDate=2023-01-01&endDate=2023-01-31- Archive data for January 2023
/archive/:code GET Get archived data for a specific stock code between dates (query params: startDate, endDate) http://localhost:3000/archive/GENNEXT?startDate=2023-01-01&endDate=2023-01-31 - Archive data for GENNEXT in January 2023
